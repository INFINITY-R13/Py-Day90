# The Most Dangerous Writing App

A web application that forces you to keep writing - stop typing and your work disappears!

## Features

- **Countdown Timer**: Configurable timeout (default 5 seconds)
- **Visual Warnings**: Editor turns red and pulses when time is running out
- **Live Statistics**: Real-time word and character count
- **Clean Interface**: Beautiful light pink theme

## Installation

1. Install dependencies:
```bash
pip install -r requirements.txt
```

2. Run the application:
```bash
python app.py
```

3. Open your browser and navigate to:
```
http://localhost:5000
```

## How to Use

1. Set your desired time limit (in seconds)
2. Click "Start Writing"
3. Keep typing continuously
4. If you stop typing for longer than the time limit, all your text will be destroyed!

## Tech Stack

- **Backend**: Flask (Python)
- **Frontend**: HTML, CSS, JavaScript
- **Styling**: Custom CSS with gradient backgrounds

## Purpose

This app is designed to help overcome writer's block by creating pressure to keep writing without overthinking or editing. It's inspired by the concept that sometimes the best way to write is to just keep going.
